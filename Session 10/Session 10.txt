Event Propagation 

    when user clicks an element in dom there are element going through 3 phases

1.Phase 1 (top to bottom)
window => document => html => body => .....=> target

2 phase 2 (target phase ) 
    which elements fire that are calles target phase 

3 phase 3 Bubbling phase (down to up)

target => parent => ......=> body => html => document => window 

By default event is bubbling event (click,input,keyup,key down,submit);


Bubbling event 
    e.target = which element clicks or fire 
    e.currentTarget = parent listeners / which elements events apply 

when we want stop event bubbling
    e.stopPropagation();
